## Setup
```
$ cd desktop
$ git clone ...
$ cd node-jsforce-starter-app
$ npm install
$ npm start
```

## gulp command
```
$ gulp
```

## jsforce-ajax-proxy
```
$ cd desktop
$ git clone git@github.com:jsforce/jsforce-ajax-proxy.git
$ cd jsforce-ajax-proxy/
$ npm install
$ npm start
```
